A WebGL based 3D user interface framework.

HTML + CSS + 3D The WhiteRabbit3D is a technology that allows a standard 2D user interface to be easily laid out and positioned in 3D space. The WhiteRabbit frameowrk is an implementation using WebGL. More content can be displayed on a page and the placement makes the information more readily understandable.

For more information, but older content, see whiterabbit3d.org.
